<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User filesystem → Electron renderer | ZIP files dropped or picked by user; content is untrusted bytes |
| Renderer → CDN (first run) | HuggingFace model download; CDN domain whitelisted in connect-src |
| Electron main process → renderer | BrowserWindow.loadFile loads local file only; no remote URL loading |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | index.html CSP | mitigate | CSP meta tag locks script-src to 'self' + cdn.jsdelivr.net; blocks injected scripts |
| T-03-02 | Elevation of Privilege | electron/main.js | mitigate | nodeIntegration:false, contextIsolation:true, sandbox:true (default); renderer has no Node.js access |
| T-03-03 | Tampering | npm install (electron, electron-builder) | accept | Both packages verified in RESEARCH.md Package Legitimacy Audit: 10+ year npm registry age, 800K+/wk downloads, maintained by Electron HQ / electron-userland; no postinstall network calls |
| T-03-04 | Denial of Service | BrowserWindow WASM | mitigate | CSP 'unsafe-eval' in script-src scoped to 'self' and cdn.jsdelivr.net only; WASM enabled only for whitelisted origins |
| T-03-SC | Tampering | npm/electron install | accept | Package Legitimacy Audit in RESEARCH.md confirms [Approved] status for both packages; no [ASSUMED] or [SUS] flags |
</threat_model>
