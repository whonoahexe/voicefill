# Requirements Archive — VoiceFill v1.0

_Original: 2026-05-21 | Archived: 2026-05-22_
_All v1 requirements satisfied at milestone close._

## v1 Requirements — All Complete

### File Input & Export Parsing

- [x] **INPUT-01**: User can drop a WhatsApp export ZIP onto the upload area (drag-and-drop) — **Delivered Phase 1 (01-01)**
- [x] **INPUT-02**: User can select a WhatsApp export ZIP via a file picker button (click-to-browse fallback) — **Delivered Phase 1 (01-01)**
- [x] **INPUT-03**: User can select a folder of extracted files (for users who already unzipped manually) — **Delivered Phase 1 (01-02)**
- [x] **INPUT-04**: User can paste or load a raw `.txt` chat log without audio (parse-only mode, voice lines annotated as [Audio not available]) — **Delivered Phase 1 (01-02)**
- [x] **PARSE-01**: Tool detects WhatsApp "with media" exports and extracts `_chat.txt` + `.opus` audio files from the ZIP — **Delivered Phase 1 (01-01, 01-02)**
- [x] **PARSE-02**: Tool detects WhatsApp "without media" exports (`<Media omitted>` placeholders) and shows a clear error explaining the re-export step — **Delivered Phase 1 (01-02)**
- [x] **PARSE-03**: Tool parses voice message lines in both WhatsApp filename formats (Android `PTT-*.opus` and iOS `00000023-AUDIO-*.opus`) and matches them to audio files by basename — **Delivered Phase 1 (01-02)**
- [x] **PARSE-04**: Tool handles WhatsApp `_chat.txt` format variations: strips BOM and Unicode directional marks, detects US-locale date format (M/D/YY) — **Delivered Phase 1 (01-02)**
- [x] **PARSE-05**: Instagram JSON export is parsed and voice messages extracted — **Delivered Phase 3 (03-02)**

### Transcription

- [x] **TRANS-01**: Voice messages transcribed using Whisper in-browser via `@huggingface/transformers` v4 — no API key, no server — **Delivered Phase 2 (02-01)**
- [x] **TRANS-02**: Model `onnx-community/whisper-tiny.en` at `dtype: 'q8'` (~40MB); inference in Web Worker — **Delivered Phase 2 (02-01)**
- [x] **TRANS-03**: First-use progress bar with explicit "~40MB — downloads once, then cached" warning — **Delivered Phase 2 (02-01, 02-02)**
- [x] **TRANS-04**: Per-message progress indicator "Transcribing 4 of 9 voice messages..." — **Delivered Phase 2 (02-01, 02-02)**
- [x] **TRANS-05**: Silence/near-silence detected via RMS energy check, skipped and annotated as [No speech detected] — **Delivered Phase 2 (02-01)**
  - *Note: no_speech_prob gate dropped — not exposed by transformers.js pipeline API; RMS gate used instead*

### Error Handling

- [x] **ERR-01**: "Without media" export detected → friendly explanation with re-export instructions — **Delivered Phase 1 (01-02)**
- [x] **ERR-02**: Corrupt or undecodable `.opus` file → annotate as `[Audio unreadable]` and continue — **Delivered Phase 2 (02-01)**
- [x] **ERR-03**: Voice message line with no matching audio file → annotate as `[Audio file missing]` — **Delivered Phase 1 (01-02)**
- [x] **ERR-04**: Audio file in ZIP with no matching voice line → silently ignored — **Delivered Phase 1 (01-02)**

### Output

- [x] **OUT-01**: Reconstructed chat log with `[Voice message: "...transcript..."]` inline — **Delivered Phase 1 (01-03)**
- [x] **OUT-02**: Summary header "X of Y voice messages transcribed" — **Delivered Phase 1 (01-03), updated Phase 2 (02-01)**
- [x] **OUT-03**: Copy full reconstructed chat to clipboard with one click — **Delivered Phase 1 (01-03)**
- [x] **OUT-04**: Download reconstructed chat as `.txt` file — **Delivered Phase 1 (01-03)**

### Design & UI

- [x] **UI-01**: Warm parchment aesthetic — background `#f5f0e8`, text `#2c2016`, accent `#8b5e3c`, CSS noise/grain texture — **Delivered Phase 1 (01-01)**
- [x] **UI-02**: Typewriter/monospace typography (Courier Prime) — **Delivered Phase 1 (01-01)**
- [x] **UI-03**: Clean and focused UI — one primary action at a time — **Delivered Phase 1 (01-01, 01-03)**
- [x] **UI-04**: Visually calm progress states — **Delivered Phase 2 (02-02)**

### Distribution (Electron)

- [x] **DIST-01**: Packaged as Electron desktop app — no HTTP serving required — **Delivered Phase 3 (03-01)**
- [x] **DIST-02**: Runs fully offline after initial Whisper model download — **Delivered Phase 3 (03-01)**
